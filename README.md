# gochat
> 最近在自学golang 语言，想用它做个即时通讯的项目

### 用到的库
github.com/gorilla/websocket

github.com/gin-gonic/gin

gorm.io/gorm

go get github.com/dgrijalva/jwt-go

### 其它
前端使用vue3实现，前后端分离开发，有一层nginx做转发和静态文件的处理
